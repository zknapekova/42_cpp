# CPP02

In this project I implemented a fixed-point number class using Orthodox Canonical Form. Based on the instructions it uses 8 fractional bits, storing values as real_value << 8.

The class supports operator overloading for arithmetic operators, comparison operators and increment/decrement operators and it's used to solve a simple BSP problem. The task was to create a function that verifies if a point lies strictly inside the triangle. As instructed, points on edges or at vertices are considered outside.
